# Complaint / Issue Tracking System (Backend)

A backend CRUD application built using **Node.js, Express, TypeScript, and MongoDB**, following **OOP principles** 

---

## Features

- Create, Read, Update, Delete complaints
- Search complaints by title or description
- Filter complaints by status, category, and priority
- Sorting and pagination support
- Input validation
- Centralized error handling
- Clean OOP-based structure (Controller → Service → Model)

---

## Tech Stack

- Node.js
- Express.js
- TypeScript
- MongoDB + Mongoose
